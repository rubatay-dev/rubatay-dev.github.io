KitaTrack Demo API

Backend Architecture

A Web API built with .NET 10 for e-wallet transaction management. This project serves as a showcase for modern engineering patterns designed for scalability, maintainability and data security.

Architecture And Patterns

  • N-Tier Architecture: Strict separation of concerns (Presentation, Application, Infrastructure).
  • SOLID Principles: Implementation of decoupled, interface-driven logic to ensure high maintainability and testability.
  • Security First: JWT Authentication with BCrypt hashing and resource-level authorization.
  • Result Pattern: Implementation of functional error handling to ensure a predictable, exception-free business logic flow.
  • Repository Pattern: Abstracting data access using Entity Framework Core for cleaner service logic.
